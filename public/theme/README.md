# Silkroad screenshot theme assets

This UI is now wired for a Silkroad Online/private-server look with real screenshot slots. Put your own in-game screenshots in this folder using these exact names:

| File path | Used by | Recommended size |
|---|---|---|
| `hero.jpg` | Homepage hero screenshot | 1920×1080 or 1920×700 JPG/WebP |
| `ingame-hotan.jpg` | Showcase card: town/trade/caravan | 1280×720 JPG/WebP |
| `ingame-unique.jpg` | Showcase card: unique boss/party drop | 1280×720 JPG/WebP |
| `ingame-jobwar.jpg` | Showcase card: PvP/job war/fortress | 1280×720 JPG/WebP |
| `logo.png` | Optional brand logo override | 64×64 transparent PNG |
| `favicon.ico` | Browser tab icon | 32×32 icon |

Fallback SVGs are included so the site still looks polished before you add screenshots:

- `fallback-hero.svg`
- `fallback-hotan.svg`
- `fallback-unique.svg`
- `fallback-jobwar.svg`

## Best screenshot choices

Use bright, busy, recognizable Silkroad Online scenes:

- Hotan/Jangan/Constantinople city shots
- Caravan/job route screenshots with traders, hunters, or thieves
- Unique boss hunts such as Tiger Girl, Cerberus, Isyutaru, Lord Yarkan
- Fortress war, arena PvP, cape fights, or guild gatherings

## Licensing note

The CMS does not ship with copyrighted Silkroad Online screenshots. Add only images you have the right to use for your server/community.
